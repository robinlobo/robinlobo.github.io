Here are **separate tables for each B1 grammar/vocabulary category**, cleanly organized so you can study and practice each area independently.

# 📘 **1. Past Tenses (Imperfect vs Indefinido)**

| **#** | **English Sentence**                                 | **Hint**                                          | **Spanish (You fill)** | **Explanation**                                         |
| ----- | ---------------------------------------------------- | ------------------------------------------------- | ---------------------- | ------------------------------------------------------- |
| 1     | When I was younger, I used to read every night.      | Habitual past → imperfect                         |                        | Repeated past actions → _imperfecto_.                   |
| 2     | Last summer we traveled to Portugal.                 | Completed event → indefinido                      |                        | Single finished action → _pretérito indefinido_.        |
| 3     | He was studying when the phone rang.                 | Background → imperfect; interruption → indefinido |                        | Ongoing action (imperfect) + interruption (indefinido). |
| 4     | I always watched movies with my brother on weekends. | Habitual past → imperfect                         |                        | Past routine → _imperfecto_.                            |
| 5     | Suddenly, the lights went out.                       | Short, finished event → indefinido                |                        | Sudden completed action → _indefinido_.                 |

# 📘 **3. Por vs Para**

| **#** | **English Sentence**                    | **Hint**         | **Spanish (You fill)** | **Explanation**                       |
| ----- | --------------------------------------- | ---------------- | ---------------------- | ------------------------------------- |
| 11    | I bought this gift for my mother.       | Recipient → para |                        | _Para_ expresses purpose/destination. |
| 12    | I’m going through the park to get home. | Route → por      |                        | _Por_ = movement/pathway.             |
| 13    | I need this done by tomorrow.           | Deadline → para  |                        | _Para mañana_ = deadline.             |
| 14    | Thank you for your help.                | Reason → por     |                        | _Por_ = cause/motivation.             |
| 15    | We stayed in Madrid for two weeks.      | Duration → por   |                        | _Por dos semanas_.                    |
