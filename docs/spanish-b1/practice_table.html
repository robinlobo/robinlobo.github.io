Practice Table

Here are separate tables for each B1 grammar/vocabulary category, cleanly organized so you can study and practice each area independently.

📘 1. Past Tenses (Imperfect vs Indefinido)

#English SentenceHintSpanish (You fill)Explanation
1When I was younger, I used to read every night.Habitual past → imperfectRepeated past actions → imperfecto.
2Last summer we traveled to Portugal.Completed event → indefinidoSingle finished action → pretérito indefinido.
3He was studying when the phone rang.Background → imperfect; interruption → indefinidoOngoing action (imperfect) + interruption (indefinido).
4I always watched movies with my brother on weekends.Habitual past → imperfectPast routine → imperfecto.
5Suddenly, the lights went out.Short, finished event → indefinidoSudden completed action → indefinido.

📘 2. Subjunctive (Wishes, Doubt, Emotion, Future)

#English SentenceHintSpanish (You fill)Explanation
6I want you to come earlier tomorrow.“Quiero que…” → subjunctiveSubjunctive after wishes.
7It’s sad that they don’t call anymore.Emotion → subjunctiveAfter es triste que, use subjunctive.
8I don’t think he knows the answer.Doubt → subjunctiveAfter no creo que, use subjunctive.
9I will call you as soon as I arrive.“Cuando” with future → subjunctiveFuture + cuando → subjuntivo.
10Before I moved here, I wanted them to help me.Wanted → subjunctiveQuería que me ayudaran.

📘 3. Por vs Para

#English SentenceHintSpanish (You fill)Explanation
11I bought this gift for my mother.Recipient → paraPara expresses purpose/destination.
12I’m going through the park to get home.Route → porPor = movement/pathway.
13I need this done by tomorrow.Deadline → paraPara mañana = deadline.
14Thank you for your help.Reason → porPor = cause/motivation.
15We stayed in Madrid for two weeks.Duration → porPor dos semanas.

18I didn’t see them at the party.Direct → los/lasNo los/las vi.
19They told us the truth.Indirect → nosNos dijeron la verdad.
20I’m going to explain it to them.Double → se + lo/laSe lo voy a explicar.
